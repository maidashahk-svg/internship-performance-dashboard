# internship-performance-dashboard
Power BI dashboard analyzing internship completion, department-wise performance, duration trends, and mentor interaction scores.

## Project Overview
This Power BI dashboard analyzes internship performance using:
- Internship completion vs dropout
- Department-wise intern distribution
- Internship duration trends
- Average mentor interaction score
